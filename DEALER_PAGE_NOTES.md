# Dealer page package

## Files in this package
- `dealer.html`
- `dealer.css`
- `dealer.js`
- `dealer-assets/` with the logo files used in the bridge band

## Upload path
Upload these files into the same repo root that already contains:
- `index.html`
- `styles.css`
- `script.js`
- `assets/`

This dealer page is designed to **use your existing `styles.css` and existing `assets/` folder**.

## Expected live URL
If you upload `dealer.html` to the repo root, the page should be available at:
- `/dealer.html`

Examples:
- `https://ionlybuyteslas.com/dealer.html`
- or your custom preferred route if you later rename it

## Pricing logic used on the page
The page leads with:
- **$750 flat** for dealer discovery and buying strategy
- **Quoted after discovery** for pilot sourcing support
- **Custom quoted** for ongoing store support

Reason: this keeps the entry point clear without locking you into underpriced ongoing dealer work.

## Notes about the bridge band
The bridge band is phrased as:
- "Worked with dealers and groups such as"
- plus a short note that names are shown to reflect working relationships and market familiarity, **not formal endorsements**

That wording is safer than making it sound like public endorsements.
