# I Only Buy Teslas homepage update

This patch updates the homepage to fix the information order based on the customer feedback.

## What changed

- Moved pricing clarity into the hero
- Added a compact "What I offer and what it costs" summary above the fold
- Changed the first nav label from **Services** to **Pricing**
- Changed the main hero CTA to **See pricing**
- Reframed the right-side panel to answer the three early questions:
  - what this is
  - how pricing works
  - what happens next
- Added a pricing FAQ item near the top of FAQ
- Added direct jump links from the hero summary to each service card

## Files included

- `index.html`
- `styles.css`
- `script.js` (copied through unchanged for convenience)

## Upload instructions

Replace the existing files in your repo root with the files in this folder.

The assets folder and API files do not need to change for this patch.
